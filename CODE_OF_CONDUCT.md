# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and harassment-free experience for everyone who participates in this project.

## Expected Behavior

- Be considerate and professional.
- Use inclusive and respectful language.
- Give and receive feedback constructively.
- Focus on the work and the facts, not on people.

## Unacceptable Behavior

- Harassment, threats, or abusive language.
- Personal attacks, insults, or demeaning comments.
- Discriminatory or exclusionary behavior of any kind.
- Deliberate disruption, trolling, or repeated bad-faith interaction.

## Scope

This code of conduct applies to project spaces and project-related communication, including pull requests, issues, reviews, and any other place where the project is discussed.

## Reporting

If you experience or witness behavior that violates this code of conduct, report it to the maintainers through the repository's usual private contact channel. If no private channel is available, open a GitHub issue and clearly mark it for maintainer attention.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not follow this code of conduct. Repeated or severe violations may lead to temporary or permanent exclusion from project participation.

## Attribution

This policy is intentionally short and practical. It is inspired by common open source code of conduct norms, including the Contributor Covenant.
